## ファームウェアのインストール
まずは動作ソフト（ファームウェア）をインストールします。
PCに接続するとRPI-RP2というドライブとして認識されます。
img

こちらのファイルをダウンロードしてドラッグアンドドロップしてください。
- uf2

自動的にRPI-RP2ドライブが消えたらファームウェアのインストール完了です。
